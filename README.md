# Prerana Tech Solutions | Corporate Website

A modern, responsive, and high-performance corporate website built for Prerana Tech Solutions, an IT services and infrastructure startup. The site features a premium UI/UX design, custom CSS animations, and a serverless backend for handling customer inquiries and job applications.

## 🚀 Features

* **Premium UI/UX:** High-contrast "iPhone 17" inspired color palette (Titanium Blue & Vibrant Orange) with modern typography (Poppins).
* **Fully Responsive:** Built with Bootstrap 5.3 to ensure seamless viewing on desktops, tablets, and mobile devices.
* **Custom Animations:** Smooth hover effects, expanding borders, and floating cards.
* **Serverless Backend (No DB required):**
  * **Contact Form:** Sends user inquiries directly to a Google Sheet.
  * **Careers Form:** Converts PDF resumes into Base64, uploads them automatically to a Google Drive folder, and logs applicant details into a Google Sheet.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Framework:** Bootstrap 5.3
* **Icons & Fonts:** FontAwesome 6.5, Google Fonts (Poppins)
* **Backend:** Google Apps Script (Google Sheets & Google Drive API)

## 📂 Project Structure

The website consists of the following interconnected pages:

* `index.html` - Homepage with hero section, company overview, and service highlights.
* `about.html` - Company mission, vision, and the "Prerana Advantage".
* `services.html` - Detailed grid of IT AMC, software development, and networking services.
* `products.html` - Hardware portfolio (CCTV, Laptops, Servers) with premium image cards.
* `Team.html` - Leadership, development, network, and field engineering team directory.
* `blog.html` - Tech insights and articles with category tags and pagination.
* `careers.html` - Job openings and the advanced application form (with PDF upload).
* `contact.html` - Location details, contact info, and the customer inquiry form.
* `style.css` - The master stylesheet controlling global branding, colors, and animations.
